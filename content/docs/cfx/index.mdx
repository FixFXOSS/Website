# Introduction to CitizenFX

CitizenFX, is the platform behind FiveM and RedM. It enables custom multiplayer servers for **Grand Theft Auto V** and **Red Dead Redemption 2**, providing a robust framework for server owners, developers, and players to create unique multiplayer experiences.

## What is CitizenFX?

CitizenFX is a platform that allows developers to create custom multiplayer servers for Rockstar Games' titles. It provides tools, APIs, and a thriving community to support server customization and resource development.

### Key Features

- **Custom Multiplayer Servers**: Host your own servers for GTA V and RDR2.
- **Extensive Modding Support**: Add custom scripts, vehicles, maps, and more.
- **Framework Integration**: Compatible with popular frameworks like ESX, QBCore, and vRP.
- **Community-Driven**: A vibrant community of developers and players contributing to the ecosystem.

## Why Use CitizenFX?

CitizenFX empowers server owners and developers to create unique experiences that go beyond the limitations of the base games. Whether you're building a roleplay server, a racing community, or a combat-focused environment, CitizenFX provides the tools you need.

### Benefits

- **Flexibility**: Customize every aspect of your server, from gameplay mechanics to visual assets.
- **Scalability**: Host servers that can handle large player counts with optimized performance.
- **Support**: Access official documentation, community forums, and third-party tools.

## Getting Started

Ready to dive in? Here's how to get started with CitizenFX:

1. **Set Up a Server**: Follow our [Getting Started Guide](/docs/cfx/getting-started) to set up your FiveM or RedM server.
2. **Explore Frameworks**: Learn about popular frameworks like [ESX](/docs/cfx/frameworks/esx) and [QBCore](/docs/cfx/frameworks/qbcore).
3. **Join the Community**: Connect with other developers and players on the [CitizenFX Collective Forums](https://forum.cfx.re).
